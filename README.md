# Scientific Calculator

A GUI-based **Scientific Calculator** built using Tkinter in Python. This calculator supports both basic and scientific operations such as trigonometric functions, logarithmic functions, square root, exponentials, and more.

## Features

- Basic arithmetic operations: Addition, Subtraction, Multiplication, Division
- Trigonometric functions: sin, cos, tan
- Logarithmic functions: log (base 10), ln (natural log)
- Exponentials and powers
- Square root (√), Pi (π), Euler’s number (e)
- Parentheses for expression grouping
- Clear (`C`) and Equal (`=`) functionality
- Error handling for invalid inputs
